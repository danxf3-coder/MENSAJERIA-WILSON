R&B mayorista - pagina web

Como abrirla:
1. Abrir index.html en el navegador.
2. La pagina funciona como catalogo con carrito y envio del pedido por WhatsApp.

Como cambiar el numero de WhatsApp:
1. Abrir app.js.
2. Buscar esta linea:
   const WHATSAPP_NUMBER = "5491123456789";
3. Reemplazar el numero por el WhatsApp real, con codigo de pais y area, sin espacios ni signos.

Como editar productos:
1. Abrir app.js.
2. Buscar la lista llamada products.
3. Cambiar nombres, codigos, categorias, precios, colores, material y medidas.

Notas:
- Los productos actuales son de ejemplo.
- Los precios son estimados y se pueden reemplazar facilmente.
- La pagina no necesita pago online: envia el carrito armado por WhatsApp.
